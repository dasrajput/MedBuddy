# Future Enhancements for MedBuddy Application

## Future Enhancements

### Active Reminders Section Improvement
- Enhance the active reminders section to provide users with a clearer overview of their upcoming medication reminders.
- Include features such as:
  - **Visual Indicators**: Use color coding to indicate the status of reminders (e.g., due, upcoming, missed).
  - **Edit Functionality**: Allow users to edit existing reminders directly from the active reminders section.
  - **Filter Options**: Provide options to filter reminders by time, medication, or status.
  - **Snooze Feature**: Implement a snooze option for reminders, allowing users to postpone notifications.
  - **History Tracking**: Show a history of past reminders and their statuses, helping users keep track of their medication adherence.

### Completed Features
- The manual entry form for medication details has been implemented, including:
  - Medicine Name
  - Number of Pills
  - Timing (Before/After Meal)
  - Meal Selection (Morning/Afternoon/Evening)
  - Reminder Time (with a time picker)
  - Validation and user feedback mechanisms are in place.

## User Interface Enhancements

### Active Reminders Section
- **Overview**: Display active reminders on the home screen.
- **Components**:
  - **Upload Image Button**: Allows users to upload an image of their medication.
  - **Manual Entry Button**: Redirects to the manual entry form.
  - **Active Reminders List**: Shows currently set reminders with options to manage them.

### Animation Feature
- **Image Upload Animation**: Show an animation when an image is uploaded successfully.
- **Redirect**: After the upload, redirect users to the set reminder page.

### Navigation
- **Bottom Navigation Panel**: Include easy access to the History page and Home screen.
